TEMA:CADENAS EN JS

Tema:cadenas

Conoce mas del tema cadenas dando click aquí

Las cadenas de JavaScript son para almacenar y manipular texto.

Para encontrar la longitud de una cadena, use la lengthpropiedad incorporada.

Los valores primitivos, como "John Doe", no pueden tener propiedades ni métodos (porque no son objetos).

Pero con JavaScript, los métodos y propiedades también están disponibles para valores primitivos, porque

JavaScript trata los valores primitivos como objetos cuando ejecuta métodos y propiedades.
